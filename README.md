## About
SIKO merupakan aplikasi web berbasis Python sebagai penunjang kegiatan transaksi toko dari barang masuk hingga keluar.

## Features
- Diagram laporan omset bulanan

## Requirements
- Python >= 3.7
- Python modules: Flask, Flask-Login, Flask-Migrate, PyTZ
